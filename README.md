# course pro Front-end

[UI 設計 (figma)](https://www.figma.com/file/3Uy156OSTesry8QePbLdfG)

[course pro Back-end (Developer only) ](https://github.com/ChengHung-Wang/course-pro-api)

為了確保使用者的隱私與安全、系統未來的發展，course pro 選擇不將後端源碼公開。

## 聲明

本系統現階段與台科校方無關，不過未來可能會與校方接洽合作。

我們會收集您的 NTUST SSO 帳密與 NTUST Email 用做驗證您的身份、訪問個人信息，course pro 有義務保護、不公開您的個人訊息，包括但不限於：您的電話、真實姓名、性別、出生日期、可連結、對應到個人的成績資料。

## 簡介

course pro 是專門開發給台科學子的一個系統，目的是為了讓您在台科有更棒的學習體驗。
我們希望你能在這個系統輕鬆地完成在學校課業上的大小事。

<details>
   <summary>查找 App 介紹</summary>

   #### 找不到合適的空間跟同學討論東西、空堂待哪裡？
   系統整合**國際大樓 12 樓討論小間、圖書館討論室、還有空教室查詢系統**，討論東西、空堂時，一查便知道可以待在哪裡！
   
   #### 頭殼版上的<strong class="text-primary">免費食物</strong>你還在一次又一次的錯過嗎？
   
   我知道你總是懊惱那麼晚才看到這個好看，正當你想衝去現場時，才發現留言處寫「已發完」。我也知道你不會想為了免費食物開啟頭殼版的貼文通知。
   
   所以，我們推出<strong class="text-primary">免費食物雷達</strong>，結合<strong class="text-danger">自然語言學習技術</strong>在偵測到贈送食物相關的貼文時，才會向你發出通知，讓你不錯過這樣的好機會。
   
   #### 東西突然丟在哪個教室找不到好煩惱...
   
   歡迎來到失物招領平台尋找，依照時間、地點、種類等等，讓尋找更加方便，一秒找回心愛的東西。
   也可以用設定提醒功能，有新的東西被刊登，馬上就可以注意到！
   刊登別人遺落的東西，拾金不昧，還可以獲得台科幣換取更多功能，一舉兩得！
   
   
   #### 想要找<strong class="text-primary">二手書</strong>、課本買完只能在旁邊生灰塵墊泡麵...
   
   每一次新的學期都要買一本課本真的好貴...
   學期結束也不曉得課本要丟去哪裡，只好讓他在旁邊生灰塵
   我們整合了 FB 的台科二手書社團，建立優良的查詢系統，有沒有學長姐在賣二手書，一查便知！

## 開發進度

### 第二階段：(預計 2023/05 完工)

#### 課程：
- 課程規劃師：自動根據您的系所、雙主修、輔修等等身份，綜合您的成績資料幫您自動分析你需要的課，你還缺的課。
- 期末評量小助手：只填你想填的，剩下的交給我們。
- 課程轉讓：讓機會落在對的人手裡，把課給你想給的人。
- 糞課雷達: 針對歷屆 GPA 過低的課程，在您的課表、課程搜尋結果增加顯眼的提示，並為您推薦更棒的替代課程。

##### 課程指南：
- 課程評價：
- 授課老師評價：
- 考古題分享:
- 課程搜尋：

##### 系統：
- 台科幣：用於使用系統某些服務的籌碼，你可以對系統貢獻來獲得台科幣。
- 多語言支持：貢獻你的母語或是你擅長的語言，讓 course pro 更完善，並在其中獲得台科幣。
- 增強 SEO 能力

### 第三階段 (2023/09 或更久)

##### 重點特色：
1. 加入人工智慧(機器學習)，
2. 手機版支援

其他功能：
- 我不想看到的同學: 設定不要跟某位或某些同學一起上課，當偵測到這位或這些同學也選了在您的課表或選課清單中的課時，系統會提醒您。
- 糞課雷達：<strong>從大家的評價、歷屆 GPA，整合 AI 運算</strong>為您過濾掉糞課，並自動根據您選課的偏好推薦您適合的課程。
- NTUST-Email
- 台科幣

1. 複製專案目錄下的`.env.example`檔案，將其貼在專案資料夾下，並將名稱改為`.env`
2. 更改 API-Endpoint, 如果你不具備 API 開發資格請使用這個[連結](https://api-course.ntust.pro/)

```shell
npm install
npm run serve
```
